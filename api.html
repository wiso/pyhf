Exceptions¶

Various exceptions, apart from standard python exceptions, that are raised from using the pyhf API.

InvalidInterpCode InvalidInterpCode is raised when an invalid/unimplemented interpolation code is requested.
InvalidModifier InvalidModifier is raised when an invalid modifier is requested.
