pyhf.set_backend¶

pyhf.set_backend(backend, custom_optimizer=None)[source]¶

Set the backend and the associated optimizer

Parameters:backend – One of the supported pyhf backends: NumPy, TensorFlow, PyTorch, and MXNet
Returns:None

Example

>>> import pyhf
>>> import tensorflow as tf
>>> pyhf.set_backend(pyhf.tensor.tensorflow_backend(session=tf.Session()))
